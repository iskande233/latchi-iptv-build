## ما تم في هذه الجولة

### 1) Google Script
- تم تجهيز سكريبت موحد مصحح يدعم:
  - تفعيل بالكود.
  - M3U.
  - Xtream عبر server/username/password.
  - MAC/Stalker عبر `source_type=mac`, `portal_url`, `mac_address`.
  - Dashboard admin actions.
  - App update عبر `get_app_update` و `set_app_update`.
- تم حفظ السكريبت داخل:
  - `google_apps_script_activation.gs`
  - `extracted_project/google_apps_script_activation.gs`

### 2) Activation API داخل التطبيق
- تم تحديث `ActivationConfig.ACTIVATION_API_URL` إلى رابط Google Script الجديد.
- تم دعم `source_type=mac` بحيث يتحول داخلياً إلى رابط `mac://stalker?...`.
- Xtream مازال يتحول إلى M3U URL بالطريقة القديمة.

### 3) ServerSyncManager
- موجود ومربوط في Home و ChannelList.
- تم إضافة `ServerHealthChecker`.
- عند تغير السيرفر، التطبيق يتحقق من صحة الرابط قبل اعتماده.
- إذا السيرفر Offline لا يمسح الكاش ولا يبدل الرابط.
- يدعم MAC source بدون HEAD عادي.

### 4) Player safe sync
- تم إضافة `PlayerServerSyncHelper`.
- `PlayerActivity.onResume()` يتحقق من السيرفر بطريقة آمنة.
- إذا تغير السيرفر أثناء المشاهدة: تظهر Overlay ثم يرجع Home.
- لم يتم تغيير محرك المشغل الأساسي.

### 5) App Update System
- `UpdateChecker` أصبح يقرأ من Google Script وليس GitHub raw.
- عند توفر تحديث يعرض Badge في Home.
- الضغط يبدأ تحميل APK عبر Android DownloadManager.
- بعد التحميل يحاول فتح مثبت APK.
- أضيفت صلاحية `REQUEST_INSTALL_PACKAGES`.

### 6) MAC/Stalker
- تم إضافة `StalkerHelper`.
- يدعم محاولة handshake + get_profile + genres + channels.
- ملاحظة: بروتوكول MAC يختلف حسب السيرفر، لذلك قد يحتاج اختبار بحساب حقيقي.

### 7) Gemini/Voice
- تم إصلاح كسر البناء الناتج عن حذف `GeminiVoiceController`.
- أضيفت نسخة آمنة بدون API Keys، تعتمد على parser محلي.
- تم إلغاء Gemini العائم العام من `FloatingBackHelper`.
- Gemini يبقى فقط في Home الهاتف، ويُخفى في TV برمجياً.

### 8) Settings
- تم إضافة زر “تحديث السيرفر الآن” في Phone Settings برمجياً.
- تم إضافة “آخر تحقق من السيرفر”.
- تم إضافة “حالة السيرفر Online/Offline”.
- تم تعليم اللغة والمشغل النشط بعلامة ✓ ولون ذهبي.
- TV Settings حصل على خيار Server Sync.

### 9) TV Focus
- تم إضافة `focusMode` في `ChannelListActivity`.
- عند فتح الفئات: يتم تعطيل Focus القنوات بـ `FOCUS_BLOCK_DESCENDANTS`.
- عند غلق الفئات: يرجع Focus القنوات بـ `FOCUS_AFTER_DESCENDANTS`.
- Back داخل الفئات يغلق الفئات فقط.

### 10) TV Live Preview
- تم إضافة `TvLivePreviewActivity`.
- في التلفاز فقط، الضغط على قناة Live يفتح Preview بدل Fullscreen مباشرة.
- Preview يعرض القناة وقائمة قنوات نفس الفئة من الكاش.
- OK على المعاينة يفتح `PlayerActivity` fullscreen.

- تجربة DownloadManager على Android TV/Phone.

## كيف يتم تحديث APK للمستخدمين بعد Codemagic
بعد نجاح Build وأخذ رابط APK، نستدعي:

`WEB_APP_URL?action=set_app_update&secret=LatchiAdmin2026&version_code=VERSION_CODE&version_name=VERSION_NAME&apk_url=APK_URL&force_update=false`

ثم التطبيق يلقط التحديث عبر:

`WEB_APP_URL?action=get_app_update&version_code=CURRENT_VERSION`
